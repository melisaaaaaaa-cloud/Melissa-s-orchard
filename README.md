## Description

This is a website that shows the beauty that lies in orchards and just normal gardens.The site contains just a few of my favourite photos captured at the glimpse of time, information about the photographer and a contact section where people can reach out.

## Project Setup Instructions

1. Clone the repository:
   git clone git@github.com:melisaaaaaaa-cloud/Melissa-s-orchard.git

2. Open the folder in Vs Code

3. Open index.html using Live Serve
